# Guess The Word Game

A simple console word guessing game written in Python.

## Description

The program picks a random word from a small list.

The player can either play normally by entering letters or use the auto play mode.

Correct guesses reveal the matching letters, while wrong guesses reduce the number of remaining lives.

The game ends when the word is fully guessed or when the player runs out of lives.

At the end of a game, the program returns to the mode menu so the user can start another game, use auto play, or quit.

## Features

- Normal play mode
- Auto Play mode
- Random word selection
- Hidden word display with underscores
- Input validation
- Guessed-letter tracking
- Wrong-letter tracking
- Win / lose detection
- Return to mode menu after each game

## Requirements

- Python 3

## How to run

    python3 main.py

When the program starts:
- enter 1 for normal play
- enter 2 for auto play
- enter 3 to quit

After a game ends, the program returns to the menu.
