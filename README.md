# Guess The Word Game

A simple console word guessing game written in Python.

## Description

The program picks a random word from a small list.  
The player guesses one letter at a time. Correct guesses reveal the matching letters, while wrong guesses reduce the number of remaining lives. The game ends when the word is fully guessed or when the player runs out of lives. At the end, the player can choose to play again.

## Features

- Random word selection
- Hidden word display with underscores
- Input validation
- Guessed-letter tracking
- Wrong-letter tracking
- Win / lose detection
- Replay support

## Requirements

- Python 3

## How to run

```bash
python3 main.py
